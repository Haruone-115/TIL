# `\n` を改行に置換したい話

AIに mermaid とかを書かせるとブロック内の文字を改行するために `\\n` とテキストを書き込んでくる
notion の レンダリングだと普通にテキストで `\\n` が表示されてしまう
簡単に改行したいな〜〜〜〜

vscode の 置換で、 `\\n` を `\n` に改行してみる？ -> ただテキストが入れ替わるだけ

うーーん😔

💡

正規表現を使うとうまくできた
検索欄の正規表現をon -> `\\\\n` を `\n` に置換！
改行されるよ！
