openAI の SDK を使うと API 叩くのがめっちゃ簡単

```ts
const res = await openai.responses.create({
  model: "gpt-5.1", // 推論モデル
  reasoning: { effort: "medium" }, // エージェント的な深めの推論

  tools: [
    {
      type: "web_search_preview",
    },
  ],
  tool_choice: "auto", // 必要に応じて web_search を自動利用

  input,
} as any);
```
